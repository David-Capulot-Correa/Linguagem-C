💻 Projetos em Linguagem C

Este repositório contém uma coleção de programas desenvolvidos em linguagem C, com foco em aprendizado, prática de lógica e estruturas fundamentais da programação.

Os códigos incluem:

🧮 Algoritmos básicos
🔁 Estruturas de repetição (for, while, do-while)
🔀 Estruturas condicionais (if, else, switch)
📦 Funções
🧠 Vetores e matrizes
🗂️ Manipulação de arquivos
🧩 Exercícios de lógica

Compile o código:

Execute:
./programa
